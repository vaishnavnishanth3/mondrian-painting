# Mondrian Painting (CSS Grid)

A simple HTML/CSS project that recreates a Piet Mondrian-style painting using pure CSS Grid layout.

## Features

- Built with only `HTML` and `CSS`
- Uses CSS Grid for precise block placement
- Follows a classic Mondrian color palette
- No external libraries or frameworks

## Project Structure

```
.
├── index.html
└── README.md
```

## How to Run

1. Clone or download this repository.
2. Open `index.html` directly in your browser.

No build tools or package installation are required.

## Color Palette Used

- Red: `#E72F24`
- Blue: `#004592`
- Yellow: `#F9D01E`
- White: `#F0F1EC`
- Black: `#232629`
- Grid gap/background: `#000000`

## Learning Focus

This project is useful for practicing:

- CSS Grid row and column sizing
- Grid item spanning with `grid-row` and `grid-column`
- Visual layout recreation from reference dimensions

## Customization

You can experiment by editing values in the `.container` styles inside `index.html`:

- `grid-template-rows`
- `grid-template-columns`
- `gap`
- block colors/classes
